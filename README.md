# MemeX: 多巴胺与安全感并存的 AI 基金经理

一个基于 OKX Agent Trade Kit 构建的，专注于【高波动 Meme 币抓取】与【无风险赚币理财】动态平衡的单体智能体。
**本项目为 #OKXAI松 龙虾赛道 开发组参赛作品。**

---

## 💡 产品简介与痛点解决

### 为什么需要 MemeX？
当前的加密市场面临一个极端的撕裂痛点：
1. **散户的 FOMO（错失恐惧）**：每天诞生数万个 Meme（土狗）币，散户极度渴望参与高波动的“多巴胺博弈”，但往往因为仓位管理失控导致爆仓或本金归零。
2. **理财工具的闲置**：用户在交易所闲置的 USDT 往往忘记投入理财产品，资金利用率极低。
3. **缺少正向的情绪反馈**：冷冰冰的交易数据无法给用户提供类似“游戏抽卡”的情绪价值。

### MemeX 的解决方案：2/8 赛博杠杆定律
MemeX 不是一个单纯的喊单机器人，而是一个**自带情绪价值的资产配置引擎**。
当你告诉它想冲刺某个 Meme 币时，它会在后台自动执行一套极严苛的纪律：
* **20% 多巴胺仓位**：自动调用 OKX Trade API 市价买入指定土狗币，让你不错过暴富机会。
* **80% 绝对防御仓位**：自动调用最新的 **OKX Earn（智能赚币）API**，将剩余资金秒级存入 USDT 活期，吃满稳定年化。
* **数字好运卡反馈**：交易完成后，根据标的风险，向用户颁发虚拟的“赛博资产卡”（如 SSR 级激战卡 / N 级休眠卡）。

## 🛠️ 技术架构与 OKX Trade Kit 集成

MemeX 采用极简的单 Agent 架构，深度集成 OKX OnchainOS 能力：

* **大脑 (LLM)**：负责意图识别、仓位数学计算、赛博朋克风格话术生成。
* **手脚 (OKX Agent Trade Kit)**：
  * `okx-market-data`：查询代币实时价格与热度。
  * `okx-trading`：执行 20% 仓位的 Meme 币市价买入。
  * `okx-earn` *(核心亮点)*：执行 80% 仓位的活期申购，实现全自动理财。
